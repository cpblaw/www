Accessibility

Carter Perry Bailey LLP endeavour to make this site accessible to as many people as possible. To that end we have ensured that the site achieves the W3C Web Accessibility Initiative Priority 1 criteria.

Web coding standards

We have also kept to a number of coding standards set out by the W3C which mean that the site should be easily visible on a number of different platforms.
